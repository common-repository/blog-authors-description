== Description ==

Provides a stupidly simple way to show informations about blog author/authors on the sidebar along with a picture and several social network links.

**Before marking this plugin as broken**, ask for some help or on the [plugin's url](http://blog.akademx.ro/blog-authors-description/).

== Installation ==

You can use the "Install Plugins" page in your dashboard to search for and automatically install the plugin.

If you need or want to manually install the plugin:

1. Download and extract Blog Authors Description
2. Upload the `blog_authors_1.2` folder to the `/wp-content/plugins/` directory
3. Activate/Network activate the plugin through the 'Plugins' menu in WordPress
3. Drag-and-drop the widget were you want to see it in Widget section of Dashboard

If you receive any errors, you can ask about them on the [plugin's url](http://blog.akademx.ro/blog-authors-description/).
